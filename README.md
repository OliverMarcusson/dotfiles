### 🗄️ Usage:
**Makes use of GNU Stow.**
```bash
# If not already installed:
sudo pacman -S stow
# or
sudo apt install stow -y

git clone git@github.com:OliverMarcusson/dotfiles ~/
cd ~/dotfiles 
stow .
```
